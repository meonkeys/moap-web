Open Source Software
for Microfinance

The MOAP Project is a new initiative to define and develop open source standards, software and documentation for microfinance institutions (MFIs) worldwide.

Specifications, examination of existing systems, and initial data models have been developed. The project will involve developers from many countries and will be conducted initially in English. Read More

MOAP seeks volunteer developers with Java and J2EE development experience, knowledge of transactions systems, and experience with ArgoUML, Http Unit, Eclipse, and jboss. If you have any of these skills, and an interest in making a difference, join the mailing list.
